Linux Lab 5: Regular Expressions

This lab will give you some experience working with regular expressions. These are useful not just in the terminal, when working with commands like grep, but in other programming languages like Python, and PHP, whenever searching through text must be done.

Assignment

  1. Visit https://regexone.com/.
  2. Complete Lessons 1 through 15 of the Interactive Tutorial.
  3. Complete Practice Problems 1-6.
  4. Place each of your regular expressions into the quotes in the return statement of each Python function in the starter code. E.g., for exercise 1, the starter code looks like: 
    def exercise1():
    # 1. Matching Decimal Numbers:
    return "" # place your pattern in these quotes
    Once you have tested your expression through RegexOne, place your expression inside the quotes. Do this for the first 6 exercises. (I encourage you to practice by completing 7 and 8 too, but these are not graded.)
  5. Submit your filled-out starter code to Moodle.

Submission

For this assignment, you only need to submit solutions for practice problems 1-6. When you do, place your patterns in the quotes in the appropriately named function inside the starter code, then remove any unused functions, and submit to Moodle.
