Max Developer Grades

Show the maximum grade (e.g. "5.11b") established by each developer. Each developer should appear only once. Show the first and last names of each developer, along with the grade string of the maximum grade they have established.

Expected Output

Note that the order and name of the fields doesn't matter, and the order of the records doesn't matter. 

+--------------------+-------------------+-----------+
| climber_first_name | climber_last_name | grade_str |
+--------------------+-------------------+-----------+
| Unknown            |                   | 5.10a     |
| Rob                | McFall            | 5.10a     |
| Art                | Cammers           | 5.10b     |
| Jeff               | Castro            | 5.10b     |
